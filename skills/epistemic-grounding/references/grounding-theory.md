# Grounding Theory Reference

This reference adapts philosophical grounding into an operational rule for Spotlight.

Source:

- Stanford Encyclopedia of Philosophy, "Metaphysical Grounding"
  https://plato.stanford.edu/entries/grounding/#Prel

## Operational Translation

Philosophical grounding asks what makes a fact obtain, not merely what is associated with it. For investigative work, the useful translation is:

> A claim is grounded only when the cited evidence supplies the reason the claim should be believed.

This matters because evidence can be nearby without grounding the claim.

## Spotlight Rules

1. **Grounds are claim-specific.** A source can ground one claim while only contextualizing another.
2. **Grounding is element-wise.** Actor, action, date, location, amount, and relationship must each be supported or named as assumptions.
3. **Partial grounds do not justify full confidence.** A claim with one unsupported material element is partially grounded.
4. **Explanatory relevance matters.** Background that explains why a claim may be plausible is not evidence that the claim is true.
5. **No circular grounding.** A claim cannot be grounded by another claim that depends on the first claim.
6. **Grounding strength can change.** New primary records, contradictions, corrections, or better source access can upgrade or downgrade a claim.

## Practical Test

Ask:

- If this evidence disappeared, would the claim still be supported?
- Does the evidence state the claim, imply it through a clear inference, or only sit near it?
- Which exact words, fields, rows, frames, or metadata values support each material element?
- What assumptions must be true for this evidence to support the claim?
- What stronger source would remove those assumptions?

If these questions cannot be answered, the item is a lead, not a verified finding.
