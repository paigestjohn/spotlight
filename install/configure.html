01 — Mode

Frontier or Local.

Pick one. Frontier uses a hosted model — faster investigations, higher quality, but queries go to the provider. Local runs a journalism model on your own machine — private by design, nothing leaves your laptop.

02 — Agent

Pick your AI agent.

Spotlight is a skill bundle — it runs inside an agent runtime. Subscription runtimes (Claude Code, Gemini, Codex) authenticate via their own login: no API key needed. Only OpenCode requires a provider key (pay per token).

Checked live against the provider when you finish — a typo fails here, not mid-investigation.

  3. Keep Obsidian running whenever you ingest findings into the vault
